# 🏡 California House Price Prediction using Machine Learning

Welcome to the California House Price Prediction Project! This project leverages Machine Learning to predict residential property prices across California based on housing, demographic, and geographical features.

From data exploration to model deployment, this project demonstrates a complete end-to-end Data Science workflow. It includes data preprocessing, feature engineering, exploratory data analysis (EDA), model training, cross-validation, hyperparameter tuning, and real-time price prediction.

## 🚀 What This Project Does

Given information about a house such as:

* 📍 Location (Latitude & Longitude)
* 💰 Median Income
* 🏠 Housing Age
* 🛏 Total Bedrooms
* 🚪 Total Rooms
* 👨‍👩‍👧 Population
* 🏘 Households
* 🌊 Ocean Proximity

the model predicts the estimated market value of the property.
